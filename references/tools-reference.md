# 工具参考

## 重要说明

**工具列表会持续动态更新**。本文档仅列出常用工具示例，实际可用工具以 `find_tool.js` 查询结果为准。建议在每次使用前通过 `find_tool.js` 获取最新的工具列表和参数信息。

## 常用工具示例

以下是部分常用工具的参考信息：

| 查询类型 | 工具关键词 | 主要参数 |
|---------|-----------|---------|
| 企业模糊搜索 | 企业模糊搜索 | `key` |
| 企业工商信息 | 企业工商信息查询 | `entid` |
| 高管信息 | 企业高管查询 | `entid`, `pindex`, `psize` |
| 经营风险 | 企业经营风险查询 | `entid` |
| 司法风险 | 企业司法风险查询 | `entid` |

## 工具复用

同一会话中已调用过的工具，可直接使用其 `tool_id`，无需再次 find_tool。

## 获取最新工具列表

```bash
node scripts/find_tool.js "查询意图描述"
```

返回的 JSON 包含：
- 工具ID (`tool_id`)
- 工具描述 (`description`)
- 参数列表 (`parameters`)
- 匹配度评分 (`score`)
